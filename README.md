<h1>AnimationWithAngular</h1>

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 13.3.1. for learning purpose.<br>
The Goal is to make a <b>modern</b> and <b>stylish</b> functional webpage<br>
There are four stages: image layout, animation, gradients, and lighting for the final product.
